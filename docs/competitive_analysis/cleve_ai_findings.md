# Cleve.ai Competitive Analysis - Initial Findings

**Date:** 2026-02-05
**Analyst:** CSO R (Manus AI)
**Source:** https://www.cleve.ai/

## Company Overview

- **Tagline:** "Turn your life into content"
- **Target Audience:** Creators, founders, business leaders (25,000+ users)
- **Platform:** Web, iOS, Android (cross-platform sync)
- **Positioning:** "Like Cursor for writing" - AI-native notes app

## Key Features Identified

### 1. Context Engine ("Cleve's Brain")
- Learns user's voice and style from past content
- Imports from: LinkedIn, X, Gmail, Instagram (coming), TikTok (coming), Threads (coming)
- Studies: Past posts, notes, digital footprint

### 2. AI Chat Integration
- Built-in AI chat with context awareness
- Multiple AI models: ChatGPT, Claude, Gemini, Perplexity
- Model switching capability
- "Edit mode" - AI can directly edit documents

### 3. Voice Features
- "Superfast Voice" - braindump thoughts
- Voice notes with auto-summarize
- Voice-to-post workflow

### 4. Organization
- Folders in folders (nested organization)
- Drag and drop context into chat
- Brand guides storage
- Content references

### 5. Collaboration
- Sharing (beta)
- Local sync
- Obsidian import/markdown support
- Full export

### 6. Skills System (Beta)
- Save favorite prompts
- LinkedIn writing, SEO, polishing
- Skills library

## UI/UX Observations (from screenshot)

- Dark mode interface
- Three-panel layout: Sidebar (folders) | Notes list | Editor + Chat
- AI Chat panel on right side
- Quick action buttons: Edit mode, Model selector
- Personalized greeting ("How can I help, Ashvin?")
